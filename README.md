# Solana AI Kit: Crypto Legal & Compliance Skill

An open-source, production-grade legal compliance and corporate architecture routing skill designed specifically for the Solana AI Kit ecosystem.

## Problem Solved
Founders and engineers building on Solana face heavy regulatory fragmentation when navigating global entity structures, on-chain geofencing rules, and cross-border stablecoin compliance. This project fills that gap by providing a token-efficient knowledge routing tree that equips coding agents (such as Claude Code or Codex) to act as expert structural guides for Web3 projects.

## Architecture & Structure
This repository matches the precise token-efficient schema expected by the Solana AI Kit framework:
- **`skill/SKILL.md`**: The master indexing node that serves context progressively to minimize large language model prompt bloating.
- **`skill/incorporation_guide.md`**: Concrete offshore foundation layout paths (DUNA, Cayman, Swiss).
- **`skill/compliance_rules.md`**: Real TypeScript RPC gate hooks and on-chain identity tracking logic templates.
- **`skill/stablecoin_regs.md`**: Deep analytical guidelines covering European MiCA laws and US stablecoin standards.

## Installation & Integration
To submodule or integrate this compliance engine directly into a standard Solana AI Kit layout:

1. Clone or add this skill package into your local configuration path:
```bash
git submodule add https://github.com skills/crypto-legal-skill
```

2. Point your core developer agent configuration file directly to the master capability index vector path: `skills/crypto-legal-skill/skill/SKILL.md`
